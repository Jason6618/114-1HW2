1. HTTP Status Code 有哪些？怎麼分類？

Ans:
HTTP 狀態碼由三位數字組成，第一位數字決定分類，共分五類：

 **五大分類**
- **1xx 資訊回應**：請求已接收，繼續處理。  
  （例：100 Continue）

- **2xx 成功**：請求成功被處理。  
  （例：200 OK、201 Created、204 No Content）

- **3xx 重新導向**：需要進一步操作（通常是轉址）。  
  （例：301 永久轉址、302 暫時轉址、304 使用快取）

- **4xx 用戶端錯誤**：**請求有問題**（例如網址錯誤、權限不足）。  
  （例：400 錯誤請求、401 未授權、403 禁止訪問、404 找不到、429 請求過多）

- **5xx 伺服器錯誤**：**伺服器處理請求時出錯**。  
  （例：500 內部錯誤、502 閘道錯誤、503 服務不可用、504 閘道逾時）

- **成功**：200
- **轉址**：301、302
- **你錯了**：404（找不到）、403（沒權限）
- **伺服器錯了**：500（伺服器掛了）、503（忙線中）
